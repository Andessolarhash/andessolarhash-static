## 💡 Uso recomendado

### Para desarrollo web:
- Usar `hero_main.png` como imagen principal del hero
- Usar `fondo_principal.png` o `hero_background.png` como fondo de sección
- Las versiones `_mobile.png` son para responsive design

### Para marketing:
- `HOME_frame_completo.png` sirve como mockup completo
- Las imágenes de productos son ideales para catálogos
